# Fully Automated Youtube Channel

Code to run a fully automated youtube that can scrape content, edit a compilation, and upload to youtube daily.

This program is used for the Meme Caviar YouTube Channel (https://www.youtube.com/c/monkeytoad11_or_thejstone2002/)

# Instructions

1. [Download](https://github.com/nathan-149/automated_youtube_channel/archive/main.zip) the Github Repository

2. Download and install [Python](https://www.python.org/downloads/) and [pip](https://pip.pypa.io/en/stable/installing/)

3. Run `pip install -r requirements.txt`

4. Create an instagram account and follow accounts you want to scrape from

5. Open main.py in a text editor and fill in necessary information

6. Run `python3 main.py`

7. Enjoy your fully automated meme maker! :) (Note, this does not attempt to upload videos because they are locked as private if the API app has not been audited by YouTube)
